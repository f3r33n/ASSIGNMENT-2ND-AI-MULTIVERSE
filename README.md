# 🤖 AI Multiverse
THIS IS THE SECOND ASSIGMENT FOR MIRAI ( THE AI MULTIVERSE)
An upgraded AI chatbot built with **Python** and **Streamlit**, supporting multiple AI models and customizable chatbot personalities. This project demonstrates how to integrate different AI providers into a single, interactive web application.

---

## 🚀 Features

- 🤖 Multiple AI Providers
  - Google Gemini
  - Cohere Command

- 🎭 Multiple Chatbot Personalities
  - Friendly
  - Professional
  - Sarcastic
  - Muslim Imam

- ⚙️ User Customization
  - Select AI model
  - Choose chatbot personality
  - Adjust creativity level
  - Adjust response length

- 💬 Interactive Chat Interface
  - Multi-line message input
  - Loading indicator while generating responses
  - Clean response layout

- ✅ Input Validation
  - Prevents empty prompts
  - Ensures personality selection before sending

- 🔐 Secure API Key Management
  - Uses `.env` file
  - API keys are never hardcoded

- 🎨 Streamlit User Interface
  - Sidebar settings
  - Responsive layout
  - Organized sections using containers

## 📖 How It Works

1. Select an AI model.
2. Choose a chatbot personality.
3. Adjust the creativity level.
4. Enter your prompt.
5. Receive an AI-generated response.
